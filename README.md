## ğŸš€ How the Project Works

1. **Authentication**
   - User logs in using credentials.
   - Access is granted to planner and events sections.
   - Logout securely ends the session.

2. **Planner Section**
   - User enters wedding date and total budget.
   - Confirms arrangements (venue, dress, makeup, jewellery, etc.).
   - Tracks amount spent and remaining budget.
   - Displays details in a Wedding Overview card.

3. **Events Section**
   - User adds individual wedding functions with date, time, venue, and notes.
   - Events are displayed as cards.
   - Events are sorted by date and can be deleted.

4. **Countdown**
   - Automatically calculates and displays days remaining until the wedding date.
